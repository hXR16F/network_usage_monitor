# Setup
### Requirements
* Linux
* Python 3.6 or newer

### Installing
```
git clone https://github.com/hXR16F/network_usage_monitor.git
cd network_usage_monitor
pip install -r requirements.txt
```
### Running
```
python network_usage_monitor.py
```
