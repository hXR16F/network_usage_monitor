# Setup
### Requirements
* Linux (also works on [Termux](https://play.google.com/store/apps/details?id=com.termux) â€” view [screenshot](https://user-images.githubusercontent.com/48186982/115868748-34291500-a43d-11eb-9bd4-488fe52888a8.png))
* Python 3.6 or newer

### Installing
```
git clone https://github.com/hXR16F/network_usage_monitor.git
cd network_usage_monitor
pip install -r requirements.txt
```
### Running
```
python network_usage_monitor.py
```
